# Creative Testing Framework — Marcus Skill

## Déclencheur
- "teste des créatifs"
- Campagne en phase OPTIMIZE

## Méthodologie
1. Définir hypothèse (quel élément : visual, headline, CTA, audience ?)
2. Créer variantes (demander à Milo)
3. Structure test : 1 variable, budget égal, 7j min ou 1000 impressions
4. Analyser : gagnant = CTR + CVR + CPA >10% meilleur
5. Implémenter gagnant, archiver learnings

## Output
```json
{
  "test_name": "Headline A/B Test",
  "variants": 2,
  "duration": "7 jours",
  "winner": "Variant B",
  "improvement": "+23% CTR, +15% CVR",
  "learning": "Headlines avec chiffres performent mieux"
}
```
