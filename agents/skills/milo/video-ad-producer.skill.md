# Video Ad Producer — Milo Skill

## Déclencheur
- "crée une vidéo"
- Tâche production vidéo

## Méthodologie

### Script 5-8s
- 0-2s : Hook (question choc ou visuel accrocheur)
- 2-5s : Problème + Solution
- 5-7s : Preuve (chiffre, témoignage, demo)
- 7-8s : CTA clair

### Formats
- Vertical 9:16 (Reels, Stories, TikTok)
- Horizontal 16:9 (YouTube, Display)
- Carré 1:1 (Feed universel)

### Génération
Veo-3 : duration 4s ou 8s, 1080p, 30fps
Audio optionnel : ElevenLabs voix off + musique
